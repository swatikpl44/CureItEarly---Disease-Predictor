About

CureItEarly - A Disease Predictor, takes the symptoms experienced by the user as input and predicts the disease using a machine learning algorithm. This predictor has been tested and is accurate as well as efficient. This web-application could help us save time and can be useful in case of emergencies. Being in initial phase, it should not be considered as an alternative for the prevailing medicinal practices and assessments. So, it is recommended that proper consultancy must be done by the user with a professional before consuming medicines based on disease predicted by our application.
